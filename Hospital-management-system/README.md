# Hospital Management System (C++)

A console-based Hospital Management System built in C++, designed to manage patient records efficiently. This project demonstrates practical use of Data Structures (vectors, structs) for dynamic data management in a real-world scenario.

---

## Features

- Add Patient: Record new patients with details like name, ID, age, gender, and illness.  
- Display Patients: View all patient records in a clear format.  
- Search Patient: Search by patient name or ID.  
- Assign Doctor: Assign a doctor to a specific patient.  
- Delete Patient: Remove a patient record from the system.  
- Dynamic Storage: All patient records are stored in vectors for easy access and modification.  

---

## How to Use

1. Clone the repository
2. Open the project folder in a C++ IDE (VS Code, Code::Blocks, Dev C++, etc.).
3. Compile and run.
4. Follow the console menu instructions to manage patients.
