# C++ DSA Projects

This repository contains advanced C++ projects focused on Data Structures and Algorithms (DSA). Each project demonstrates practical use of vectors, arrays, structs, and dynamic data handling, simulating real-world applications and improving algorithmic thinking in C++.

---

## Projects Included

### 1. Hospital Management System
A console-based application to manage hospital patient records efficiently.  
Features:
- Add, display, search, update, and delete patient records  
- Assign doctors to patients  
- Handle dynamic data using vectors and structs  
- Input validation and error handling  

[View Project](Hospital-management-system)

---

### 2. Social Media System
A simplified social media console application to simulate interactions between users.  
Features:
- Create and manage user accounts  
- Add posts, likes, and comments  
- Follow and unfollow other users  
- Display user feeds dynamically  
- Efficient handling of dynamic data  

[View Project](Social-Media-simulator)
