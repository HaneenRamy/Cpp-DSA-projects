# Social Media System (C++)

A console-based Social Media System built in C++, designed to simulate basic social media functionalities while demonstrating the use of Data Structures (vectors, structs) for managing dynamic data.

---

## Features

- Create User Profiles: Add new users with unique IDs and names.  
- Post Messages: Users can post messages to their profile.  
- Follow/Unfollow Users: Users can follow or unfollow other users.  
- Like Posts: Users can like posts from other users.  
- View Feed: Display all posts with user info, likes, and interactions.  
- Dynamic Storage: All users and posts are managed using vectors for easy access and modification.  

---

## How to Use

1. Clone the repository
2. Open the project folder in a C++ IDE (VS Code, Code::Blocks, Dev C++, etc.).
3. Compile and run.
4. Follow the console menu instructions to interact with the system.
